# 🛒 Flask E-Commerce Website

A mini E-Commerce website developed using Python Flask, HTML, and CSS.

## 🚀 Features

- User Login & Logout
- Product Display
- Add to Cart
- Remove from Cart
- Checkout System
- Success Page
- Session Handling
- Responsive UI Design

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3

## ▶️ Run Project

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
